# Color de fondo por sección (portada)

De la maqueta de referencia. Pendiente de aplicar a la web real.

## Paleta

| Token | Hex | Uso |
| --- | --- | --- |
| `cream` | `#FBF6EE` | El más claro. Secciones de "lectura" (intro, carta). |
| `sand` | `#F3E9D8` | Base del body. Secciones neutras. |
| `sand-deep` | `#E9DAC0` | Arena tostada. Secciones que deben destacar sin ser oscuras. |
| `night` | `#0F2A3C` | Azul noche. Bloques ancla + footer. |
| `night-2` | `#143A4C` | Azul noche más claro. Dónde estamos. |
| `terra` | `#C0603A` | Acento: antesalas, precios, punto del título. |
| `gold` | `#D9A24C` | Acento sobre fondo oscuro. |
| `sea` | `#2C6E6B` | Verde mar. Solo tarjetas y citas. |

## Orden de la portada

1. **Hero** — degradado de cielo: `#0F2A3C` → `#1C4258` → `#C07A4E` → `#B56A44`.
2. **Bienvenidos** — `cream`.
3. **Un día en La Calita** — `sand-deep`.
4. **Cuatro mundos (cocina)** — `sand`. Las cuatro tarjetas sí van con color propio: `night`, `sea`, `terra`, `#3A2F27`.
5. **Los que no fallan (carta)** — `cream`.
6. **Nuestra historia** — `night`, texto blanco.
7. **Música a pie de playa** — `sand-deep`.
8. **Opiniones** — `sand`. Tarjetas en `cream` con borde.
9. **Dónde estamos** — `night-2`, texto blanco.
10. **Footer** — `night`.

## La idea

Tres tonos de arena que se alternan (crema → arena → arena oscura) para que se
note el corte entre secciones sin perder calidez, y dos bloques oscuros que
hacen de ancla: historia a mitad de página y "dónde estamos" al final, que ya
enlaza con el footer. El color saturado se reserva para las tarjetas de cocina.
